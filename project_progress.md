## 現在のフェーズ：Phase 39 (完了 - スマホの極限上寄せ ＆ メタデータ開閉安定化 ＆ 新タイムライン一行コンパクト化 ＆ 1日目制限)
### 実施内容
- **スマホの極限上部寄せ調整（Y=30px）**:
  - 拡大時のスマホのY目標座標を `60px` からさらに上方の **`30px`** へと限界引き上げ！
  - スマホの液晶天井が完全に画面最上部に届くようになり、画面の縦表示スペースを1pxの無駄もなく極限まで活用。
- **100%拡大解除を保証する「メタデータ開閉判定」＆「スクリーン空間Rect2ヒットテスト」**:
  - GDScript 2.0のラムダ式がプリミティブ型（`bool`）を「値コピー」でキャプチャしてしまい、状態更新が反映されないバグを根本解決。
  - スマホノード自体のメタデータ（`phone.set_meta("is_picked_up", ...)`）で動的に状態を管理し、ラムダ内から常にリアルタイムな状態を参照可能に。
  - さらに、Godotのローカル座標系のスケール誤差を完全に排し、画面全体の絶対座標からスマホのビジュアル領域を逆算する **「スクリーン空間Rect2ヒットテスト」** を導入。これにより、スマホの外をクリックした時のクローズ動作が100%完璧に動作することを絶対保証！
- **初日（1日目）のタイムライン制限とUX改善**:
  - 初日はまだ他プレイヤーの学習報告が存在しないため、1位・2位などのランキングや「いいね/応援」ボタンを表示しない親切設計に。
  - 代わりに、チキスタの仕組みを優しく案内する「タイムラインのお知らせ」ウエルカムカードを表示し、初日の理不尽な感情（いいねを押してライバルに得点が入る不憫さ）を完全に排除。
- **2日目以降のタイムラインの超軽量「一行コンパクト化」実装**:
  - 巨大だったスクロールフィードカードを廃止し、高さ **`52px`** の美しい一行コンパクトなリストに再編成！
  - リストには「各教科を代表するテーマ色のアバター（イニシャル付き）」、「教科名：プレイヤー名（点数）」を一行に整列。
  - 右端に極小の円形フラットボタン **「疑」**（赤基調）と **「応」**（緑基調）をスマートに配置。クリックすると即座に無効化され、ボタン自体が塗りつぶし状態へ鮮やかにTween。
  - 5つの教科すべてが一つの画面内に美しく収まり、一切スクロールを必要としない極上のアプリ感・手触りを実現！
- **Godotヘッドレスビルド検証**:
  - Webエクスポートを再度実行し、エラー皆無の「Exit Code 0」で完全クリア。

## プロジェクト構造

```
study-chikenrace/
├── project.godot              # プロジェクト設定（オートロード: Global, SceneTransition）
├── *.tscn                     # シーンファイル（Main, Title, Profile, ResultScene）
├── scripts/
│   ├── core/                  # ゲームロジック層
│   │   ├── Global.gd          #   セーブ/ロード/状態管理（オートロード）
│   │   ├── BackendManager.gd  #   スコア通信・投票API・タイムライン
│   │   ├── StudyDeck.gd       #   デッキ構築・ドロー・バースト判定
│   │   ├── GameSession.gd     #   セッション管理
│   │   ├── AudioManager.gd    #   BGM/SE再生
│   │   └── Enums.gd           #   教科・アイテム列挙型
│   ├── data/
│   │   └── CardData.gd        #   カードデータクラス
│   └── ui/                    # UI/表示層
│       ├── DeskTheme.gd       #   テーマ定数・UIウィジェットファクトリ
│       ├── GameScene.gd       #   メインゲーム画面（カバン構築〜チキンレース〜報告）
│       ├── TitleScene.gd      #   タイトル画面・チュートリアル
│       ├── ProfileScene.gd    #   プロフィール入力
│       ├── ResultScene.gd     #   最終通知表
│       └── SceneTransition.gd #   フェード遷移（オートロード）
└── assets/                    # アセット
    ├── hgrsmp.ttf             #   メインフォント
    ├── bgm_main.wav / se_*.wav #  BGM・SE
    ├── *.png                  #   UI画像群
    ├── split/                 #   アイテム・教科アイコン
    └── tutorial/              #   チュートリアルスライド
```
