## 現在のフェーズ：Phase 40 (完了 - スマホ元の位置への完全復帰 ＆ カバン画面タイムラインの完全刷新 ＆ Web文字化け排除)
### 実施内容
- **拡大時スマホ座標の絶妙な黄金比調整（Y=70pxへの微調整）**:
  - 拡大率1.32倍の際に画面上部にヘッダーがはみ出しすぎていた問題を解決するため、目標Y座標を `30px` から **`70px`** へと微調整。これにより、ノッチやヘッダーが切れるのを防ぎ、画面上下で極めて均等かつ美しい黄金比の余白バランスを実現しました。
- **スマホ元の位置（木の机の上）への100%ピクセルパーフェクトな完全復帰**:
  - `_create_smartphone_mockup` において、スマホ本体に設定されていたレイアウトアンカーをすべてクリア（`anchor_left/top/right/bottom = 0.0`）の絶対座標に統一。
  - Godotのコンテナ自動アンカー再配置エンジンとの競合を完全排除し、拡大から元に戻る際に元の位置（`Vector2(88, 300)`）に寸分の狂いもなく確実にTweenで着地する手触りを絶対保証しました。
- **カバン構築画面のスマホタイムライン完全刷新（画面UIが変わらない問題の解消）**:
  - 黒板画面（朝の会）だけが刷新され、カバン画面側が古い巨大カードUIのままになっていた問題を解消。
  - `_build_timeline_feed` を完全刷新し、1日目は温かい「ウエルカムお知らせカード」、2日目以降は高さ `52px` の超美しい一行コンパクトリスト ＋ 極小「疑/応」バッジボタン設計を完璧に統合しました。
- **フッター切り替えタブのWeb文字化け（豆腐文字）の完全回避**:
  - Webビルド時のフォント非対応による文字化けを排除するため、フッタータブのテキストから絵文字（🏠📊🎯）を完全に排除。
  - クリーンなフラットテキスト「タイムライン」「学習分析」「目標」へと換装しました。

## プロジェクト構造

```
study-chikenrace/
├── project.godot              # プロジェクト設定（オートロード: Global, SceneTransition）
├── *.tscn                     # シーンファイル（Main, Title, Profile, ResultScene）
├── scripts/
│   ├── core/                  # ゲームロジック層
│   │   ├── Global.gd          #   セーブ/ロード/状態管理（オートロード）
│   │   ├── BackendManager.gd  #   スコア通信・投票API・タイムライン
│   │   ├── StudyDeck.gd       #   デッキ構築・ドロー・バースト判定
│   │   ├── GameSession.gd     #   セッション管理
│   │   ├── AudioManager.gd    #   BGM/SE再生
│   │   └── Enums.gd           #   教科・アイテム列挙型
│   ├── data/
│   │   └── CardData.gd        #   カードデータクラス
│   └── ui/                    # UI/表示層
│       ├── DeskTheme.gd       #   テーマ定数・UIウィジェットファクトリ
│       ├── GameScene.gd       #   メインゲーム画面（カバン構築〜チキンレース〜報告）
│       ├── TitleScene.gd      #   タイトル画面・チュートリアル
│       ├── ProfileScene.gd    #   プロフィール入力
│       ├── ResultScene.gd     #   最終通知表
│       └── SceneTransition.gd #   フェード遷移（オートロード）
└── assets/                    # アセット
    ├── hgrsmp.ttf             #   メインフォント
    ├── bgm_main.wav / se_*.wav #  BGM・SE
    ├── *.png                  #   UI画像群
    ├── split/                 #   アイテム・教科アイコン
    └── tutorial/              #   チュートリアルスライド
```
